## üî® Usage

How to install

```shell
npm install -d vuepress-plugin-anchor-right
```

Use in your project

```js
// docs/.vuepress/config.js
import vuepressPluginAnchorRight from 'vuepress-plugin-anchor-right';

export default  {
  ...,
  plugins: [
    [vuepressPluginAnchorRight()],
  ]
};
```

> If you are using vuepress version v2.0.0-beta 49, then you should use vuepress plugin anchor right v0.0.1-beta 3„ÄÇ Otherwise, use the latest.
